This plugin allows you to add a Dixa Chat widget into the footer of every page on your website by hooking into wp_footer.

== Description ==

= Easily Insert A Dixa Chat Widget =

Insert a Dixa Chat widget ino every page on your website without the need to edit your theme files!

= Features =

* Quick to set up
* Simple to insert script

== Installation ==

1. Install WP Dixa Chat by uploading the `wp-dixa-chat` directory to the `/wp-content/plugins/` directory. (See instructions on <a href="http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/" rel="friend">how to install a WordPress plugin</a>.)
2. Activate Dixa Chat through the `Plugins` menu in WordPress.
3. Insert the Dixa Chat code by going to the `Settings > Dixa Chat menu.
